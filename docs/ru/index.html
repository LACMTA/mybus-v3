260

Artesia Station - Pasadena
via Fair Oaks Av, Atlantic Bl

Новое расписание
Обновление: Уточненное расписание на субботу, воскресенье и будние дни. Каждый второй рейс Line 260 в будние и выходные дни между Pasadena и South Gate через Atlantic Bl будет продлен через Imperial Highway для обслуживания ключевого регионального пункта назначения — станции линий Line A и Line C Willowbrook / Rosa Parks Station. Другие существующие рейсы Line 260 между станциями Pasadena и Artesia Station через Atlantic Bl сохранятся, но будут переименованы на Line 261. Пассажиры, которые в настоящее время ездят по маршруту Line 260 по Fair Oaks Av Atlantic Bl между Pasadena и Imperial Highway могут пользоваться линией Line 260 или 261.
Причина: Для повышения надежности, более точного отражения фактического времени в пути и улучшения согласованности расписания внутри региона.
